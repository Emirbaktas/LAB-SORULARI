# Veri Yapıları - Lab Ödevleri

Bu repo, Veri Yapıları dersinin lab ödevlerini evde tekrar çalışarak uyguladığım Java projelerini içermektedir.

## İçerik

| Klasör | Açıklama |
|--------|----------|
| `soru1/` | Bağlı liste (Node, List, Data, Main) |
| `soru2/` | Bağlı liste varyasyonu (Node, List, Data, Main) |
| `soru3/` | Bağlı liste uygulaması (Node, List, Main) |
| `soru4/` | Öğrenci kayıt sistemi (Student, StudentShowerAndChanger, Main) |
| `soru5/` | Özyinelemeli işlemler (RecursiveOperations, Main) |
| `soru6/` | Tek yönlü bağlı liste (SinglyLinkedList, Node, Main) |
| `soru7/` | Bağlı liste (LinkedList, Node, Main) |
| `soru8/` | Özel yığıt yapısı (CustomStack, ListNode, Main) |

## Kullanılan Teknolojiler

- Java
- Veri yapıları: Bağlı liste, yığıt (stack), özyineleme (recursion)

## Notlar

- Tüm kodlar lab ortamında çözülen ödevlerin evde tekrar yazılmış halidir.
- Her soru kendi klasöründe bağımsız olarak derlenip çalıştırılabilir.

## Çalıştırma

```bash
cd soruX
javac *.java
java Main
```
